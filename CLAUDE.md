# ASC public website

## Design

- **Theme: Cream Light** (decided 2026-07-09). Light cream canvas with dark text — the
  light counterpart to the admin dashboard (`web/admin`), which is permanently dark
  (charcoal canvas, cream text).
- **Direction: Field Notebook** (decided 2026-07-17, from `design/vibe-lab.html` #02).
  Graph-paper cream canvas (`#fbf8ee`) with warm ink (`#38342a`), red margin rule,
  serif body (Iowan Old Style stack), Caveat for handwritten annotations, taped-on
  cards, and stamp accents in rust `#c93d12`. Tokens live in `app/globals.css` `@theme`.
- Brand palette and typography live in `/Users/jwilli/Dev/ASC/branding/branding_info/BRANDING.md`;
  logo assets in `/Users/jwilli/Dev/ASC/branding/logos/`.
